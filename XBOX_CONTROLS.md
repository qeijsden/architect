# Xbox Controller Quick Reference

## 🎮 Controller Connected!

When your Xbox controller is connected, you'll see a gamepad icon in the game.

---

## 🎯 Game Controls

### Basic Movement
- **Left Stick** or **D-Pad** ➜ Move Left/Right
- **A Button** ➜ Jump
- **Start** ➜ Pause Game

### Editor Mode
- **Right Trigger (RT)** ➜ Place Block
- **Left Trigger (LT)** ➜ Remove Block
- **Right Bumper (RB)** ➜ Next Block Type
- **Left Bumper (LB)** ➜ Previous Block Type
- **Right Stick** ➜ Move Camera
- **Left Stick** ➜ Move Cursor

---

## 📱 Menu Navigation

### Navigating Menus
- **Left Stick** or **D-Pad** ➜ Move Selection
- **A Button** ➜ Select/Confirm
- **B Button** ➜ Back/Cancel
- **RB/LB** ➜ Switch Tabs

---

## 💫 Haptic Feedback

Feel the game through controller vibrations:

- **Light Pulse** - Checkpoint reached
- **Strong Rumble** - Death/Respawn
- **Victory Vibe** - Level complete!
- **Gentle Tap** - UI navigation

---

## 🔧 Troubleshooting

### Controller Not Working?

1. **Check Connection**
   - Reconnect controller via USB or Bluetooth
   - Press Xbox button to wake controller

2. **Check Batteries**
   - Replace batteries or charge battery pack
   - Try wired connection

3. **Try Another Controller**
   - Test with different Xbox controller
   - Verify controller works in other games

4. **Restart Game**
   - Close and reopen Architect
   - Controller should auto-detect on startup

### Controller Detected but Not Responding?

- Check if another program is using the controller
- Update controller firmware via Xbox Accessories app
- Restart your PC/Xbox

## 🌟 Tips & Tricks

### For Best Experience:
1. **Use Wired Connection** - Lower latency, no battery worries
2. **Enable Vibration** - Enhances gameplay feedback
3. **Adjust Deadzone** - If stick drift occurs (coming in future updates)
4. **Practice Makes Perfect** - Controller precision improves with practice

### Advanced Tips:
- **Quick Save**: Use checkpoints strategically
- **Camera Control**: Use right stick for better view of level
- **Block Placement**: Hold RT and use left stick for rapid building
- **Menu Speed**: Hold A to select rapidly in menus
